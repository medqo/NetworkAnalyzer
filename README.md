# Network Troubleshooting Assistant

Network Troubleshooting Assistant is a Streamlit-based tool designed to support Cisco network troubleshooting.

The application guides users through the troubleshooting process by suggesting verification commands based on observed network problems. It analyzes pasted outputs from commands such as `ping`, `traceroute`, and Cisco `show` commands, identifies possible Layer 1, Layer 2, or Layer 3 issues, and provides recommended recovery commands.

## Features

- Select network failure scenarios
- Display possible causes classified by OSI layer
- Recommend troubleshooting commands
- Analyze Cisco command outputs
- Detect VLAN, trunk, interface, routing, and OSPF issues
- Suggest recovery commands
